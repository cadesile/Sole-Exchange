The Taxonomy Token module allows the user to set vocabularies for which there will be created specific node type tokens, i.e., these tokens will generally be available under the "node" or "global" sections of available tokens list.

It's main use is together with the Pathauto module.

Configuration

All configuration can be done at Taxonomy Token settings page.

Performance impact consideration

Having the ability to not choose specific tokens for all vocabularies reduces the performance impact of this module.
